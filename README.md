# Gladius

## Stack

- **Astro 5** (static output)
- **Tailwind CSS 3** + typography plugin
- **TypeScript** (strict)
- **Vercel** (deploy)
- **Vercel Analytics** + Speed Insights

## Desenvolvimento

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # gera dist/
npm run preview  # serve dist/ local
```
